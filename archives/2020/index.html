MapReduce using the Bulk Synchronous Parallel Bridging Model and Linguistic Reflection.

The project exploits that the BSP model can be captured by specific concurrent Abstract State Machines (cASMs), which allows us also to identify algorithms that cannot be captured this way, but require some extensions, for which we explore linguistic reflection.
